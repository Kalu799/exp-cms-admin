# Exo - CMS Admin

Dashboard d'administration permettant de superviser les visiteurs présents, consulter l'historique des visites et gérer le personnel ainsi que les formations.

Structure :
- `js/config.js` : configuration du projet ;
- `js/main.js` : point d'entrée général ;
- `js/api/` : fonctions qui parlent avec WordPress ;
- `js/auth/` : gestion de la session utilisateur ;
- `js/pages/` : logique propre à chaque page ;
- `js/utils/` : petites fonctions réutilisables.
